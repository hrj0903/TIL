# VSC 간편 단축키

## 반복되는 단어 한방에 수정

⌘ + D or Ctrl + D

## 클릭하는 곳마다 커서 생성

⌥ + Click or Alt + Click

## 코드 위/아래로 움직이기

⌥ + ↑ / ↓ or Alt + ↑ / ↓

## 코드 복사해서 위/아래로 움직이기

⌥ + ⇧ + ↑ / ↓ or Alt + ⇧ + ↑ / ↓ ##코드 블록 한방에 코멘트 처리하기
⌘ + / or Ctrl + /

## 선택된 영역에 커서 만들기

⌥ + ⇧ + i or Alt + ⇧ + i

## 마우스가 가는 곳 마다 커서 만들기

⌥ + ⇧ + Mouse Drag or Alt + ⇧ + Mouse Drag

## 파일 맨 위-아래로 한번에 이동하기

Top / Bottom of File: ⌘ + ↑ / ↓ or Ctrl + Home / End

## 사이드바 숨기기

Hide / Show Sidebar: ⌘ + B or Ctrl + B
